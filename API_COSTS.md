# API 비용 분석

## ✅ 완전 무료 API들

### 1. 번역 API
- **Google Translate 무료 엔드포인트** (비공식, 무료)
  - 현재 코드에서 주로 사용
  - API 키 불필요
  - 제한 없음 (하지만 과도한 사용 시 IP 차단 가능성 있음)
  
- **MyMemory API** (Fallback, 무료)
  - 일일 10,000자 무료
  - API 키 불필요
  - 제한 초과 시 에러 반환 (유료 플랜 필요 없음)

**결론: 번역 기능은 완전 무료입니다! 💰**

### 2. Firebase (기본 사용)
- **Firestore Database**
  - 무료 티어: 읽기 50,000/일, 쓰기 20,000/일, 삭제 20,000/일
  - 대부분의 앱은 무료 티어로 충분
  
- **Firebase Authentication**
  - 무료 티어: 월 50,000명까지 무료
  - Anonymous Auth 포함
  
- **Firebase Cloud Messaging (FCM)**
  - 푸시 알림 무료 (무제한)

**결론: 소규모 앱은 Firebase도 무료입니다! 💰**

### 3. RevenueCat
- **IAP 관리 서비스**
  - 무료 티어: 월 $10,000 매출까지 무료
  - 광고 제거 기능에 사용

**결론: IAP 기능도 무료 티어로 충분합니다! 💰**

## ⚠️ 주의사항

### Google Translate 무료 엔드포인트
- 비공식 API이므로 언제든 차단될 수 있음
- 과도한 사용 시 IP 차단 가능
- **대안**: MyMemory API로 자동 fallback 처리됨

### Firebase 사용량 모니터링
- 사용자가 많아지면 무료 티어 초과 가능
- Firestore 읽기/쓰기 횟수 주의
- Firebase Console에서 사용량 모니터링 가능

## 💡 비용 절감 팁

1. **번역 API**: 
   - 현재 구조가 이미 최적화됨 (Google Translate → MyMemory fallback)
   - 추가 비용 없음

2. **Firebase**:
   - 불필요한 읽기/쓰기 최소화
   - 캐싱 활용
   - 인덱스 최적화

3. **RevenueCat**:
   - 무료 티어로 충분 (월 $10,000 매출까지)

## 📊 예상 비용

### 초기 단계 (사용자 < 1,000명)
- **번역 API**: $0 (무료)
- **Firebase**: $0 (무료 티어)
- **RevenueCat**: $0 (무료 티어)
- **총 비용: $0/월** ✅

### 성장 단계 (사용자 1,000-10,000명)
- **번역 API**: $0 (무료)
- **Firebase**: $0-25/월 (사용량에 따라)
- **RevenueCat**: $0 (무료 티어)
- **총 비용: $0-25/월** ✅

### 대규모 (사용자 > 10,000명)
- **번역 API**: $0 (무료) 또는 유료 API 전환 고려
- **Firebase**: $25-100/월
- **RevenueCat**: $0 (무료 티어)
- **총 비용: $25-100/월** ✅

## 결론

**현재 구현된 모든 기능은 무료 API를 사용하고 있습니다!** 💰

추가 비용이 발생하는 경우:
1. Firebase 사용량이 무료 티어를 초과할 때
2. RevenueCat 매출이 월 $10,000를 초과할 때
3. (선택사항) 더 정확한 번역을 위해 유료 번역 API로 전환할 때

하지만 초기 단계에서는 **완전 무료**로 운영 가능합니다! 🎉
