Privacy Policy

Last Updated: December 2, 2025

1. Introduction

TetraChat Global ("we", "our", or "the app") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and protect your personal information when you use our multilingual chat application supporting English, Japanese, Spanish, and Chinese.

2. Information We Collect

2.1 Account Information

  • Email address
  • Password (encrypted)
  • Display name (nickname)
  • Language preference (English, Japanese, Spanish, or Chinese)
  • Device ID

2.2 Chat Data

  • Messages you send and receive
  • Chat room information
  • Message timestamps

2.3 Technical Information

  • Push notification tokens
  • App usage data
  • Device information

3. How We Use Your Information

  • To provide chat services between users of different languages
  • To send push notifications for new messages
  • To translate messages between languages
  • To improve app functionality and user experience
  • To prevent fraud and ensure security

4. Third-Party Services

4.1 Firebase (Google)

We use Firebase for:

  • Authentication
  • Cloud database (Firestore)
  • Push notifications

Firebase Privacy Policy: https://firebase.google.com/support/privacy

4.2 Google AdMob

We display advertisements using Google AdMob. AdMob may collect:

  • Device advertising ID
  • IP address
  • Device information
  • Ad interaction data

AdMob Privacy Policy: https://support.google.com/admob/answer/6128543

4.3 Google Translate API

Messages are automatically translated using Google Translate API. Messages sent to the translation service are processed according to Google's privacy policy.

5. Data Storage and Security

  • All data is stored on Google Firebase servers
  • Passwords are encrypted using industry-standard methods
  • Data is transmitted over secure HTTPS connections
  • We implement security measures to protect against unauthorized access

6. Data Retention

  • Account data is retained until you delete your account
  • Chat messages are retained as long as the chat room exists
  • When you delete your account, all associated data (messages, profile, chat rooms) is permanently deleted

7. Your Rights

You have the right to:

  • Access your personal data
  • Delete your account and all associated data
  • Opt-out of push notifications in device settings
  • Request data correction

8. Children's Privacy

Our app is not intended for children under 13 years of age. We do not knowingly collect personal information from children under 13.

9. Changes to Privacy Policy

We may update this Privacy Policy from time to time. We will notify users of significant changes through the app or email.

10. Contact Us

If you have questions about this Privacy Policy, please contact us at:

Email: jihun.jo@yahoo.com

11. Data Sharing

We do not sell, trade, or rent your personal information to third parties. Data is only shared with:

  • Firebase for app functionality
  • Google AdMob for advertising
  • Google Translate API for translation services

Politica de Privacidad

Ultima actualizacion: 2 de diciembre de 2025

1. Introduccion

TetraChat Global ("nosotros", "nuestro" o "la aplicacion") esta comprometido con la proteccion de su privacidad. Esta Politica de Privacidad explica como recopilamos, usamos y protegemos su informacion personal cuando utiliza nuestra aplicacion de chat multilingue que soporta ingles, japones, espanol y chino.

2. Informacion que Recopilamos

2.1 Informacion de la Cuenta

  • Direccion de correo electronico
  • Contrasena (encriptada)
  • Nombre para mostrar (apodo)
  • Preferencia de idioma (ingles, japones, espanol o chino)
  • ID del dispositivo

2.2 Datos del Chat

  • Mensajes que envia y recibe
  • Informacion de la sala de chat
  • Marcas de tiempo de los mensajes

2.3 Informacion Tecnica

  • Tokens de notificaciones push
  • Datos de uso de la aplicacion
  • Informacion del dispositivo

3. Como Usamos Su Informacion

  • Para proporcionar servicios de chat entre usuarios de diferentes idiomas
  • Para enviar notificaciones push de nuevos mensajes
  • Para traducir mensajes entre idiomas
  • Para mejorar la funcionalidad de la aplicacion y la experiencia del usuario
  • Para prevenir fraudes y garantizar la seguridad

4. Servicios de Terceros

4.1 Firebase (Google)

Usamos Firebase para:

  • Autenticacion
  • Base de datos en la nube (Firestore)
  • Notificaciones push

Politica de Privacidad de Firebase: https://firebase.google.com/support/privacy

4.2 Google AdMob

Mostramos anuncios usando Google AdMob. AdMob puede recopilar:

  • ID de publicidad del dispositivo
  • Direccion IP
  • Informacion del dispositivo
  • Datos de interaccion con anuncios

Politica de Privacidad de AdMob: https://support.google.com/admob/answer/6128543

4.3 API de Google Translate

Los mensajes se traducen automaticamente usando la API de Google Translate. Los mensajes enviados al servicio de traduccion se procesan segun la politica de privacidad de Google.

5. Almacenamiento y Seguridad de Datos

  • Todos los datos se almacenan en servidores de Google Firebase
  • Las contrasenas se encriptan usando metodos estandar de la industria
  • Los datos se transmiten a traves de conexiones HTTPS seguras
  • Implementamos medidas de seguridad para proteger contra el acceso no autorizado

6. Retencion de Datos

  • Los datos de la cuenta se retienen hasta que elimine su cuenta
  • Los mensajes de chat se retienen mientras exista la sala de chat
  • Cuando elimina su cuenta, todos los datos asociados (mensajes, perfil, salas de chat) se eliminan permanentemente

7. Sus Derechos

Usted tiene derecho a:

  • Acceder a sus datos personales
  • Eliminar su cuenta y todos los datos asociados
  • Desactivar las notificaciones push en la configuracion del dispositivo
  • Solicitar correccion de datos

8. Privacidad de los Ninos

Nuestra aplicacion no esta destinada a ninos menores de 13 anos. No recopilamos intencionalmente informacion personal de ninos menores de 13 anos.

9. Cambios en la Politica de Privacidad

Podemos actualizar esta Politica de Privacidad de vez en cuando. Notificaremos a los usuarios sobre cambios significativos a traves de la aplicacion o por correo electronico.

10. Contactenos

Si tiene preguntas sobre esta Politica de Privacidad, contactenos en:

Correo electronico: jihun.jo@yahoo.com

11. Compartir Datos

No vendemos, comerciamos ni alquilamos su informacion personal a terceros. Los datos solo se comparten con:

  • Firebase para la funcionalidad de la aplicacion
  • Google AdMob para publicidad
  • API de Google Translate para servicios de traduccion
